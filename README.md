#This is AIMCHECK!

Train both your reflexes and your quick thinking abilities while trying to get as much points as you can!

##Controls

Click the GREEN circles to win points 
CAREFUL! if enough time passes or you click a RED circle you lose a life, lose all three and its GAME OVER

##How to Download
Download the AIMCHECK.zip File
Unzip the the .zip file and run the SMLF_AIMCHECK.exe 
Enjoy!
